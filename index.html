"Where Should We Go?" Web App

With this web application you can create a collaborative list of places you would like to visit or learn more about.
You can perform the fundamental CRUD operations (Create, Read, Update, Delete), as well as browse images, custom maps, and interesting statistics about your places on the list.
The map on the main page marks the spot of all the places on the list.
